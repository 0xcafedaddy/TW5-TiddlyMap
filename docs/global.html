Collection

A collection contains a group of elements, usually nodes (see NodeCollection) or edges (see EdgeCollection).

A collection object corresponds to a CollectionTypeString.

Type:
  • Array.<*> | Hashmap.<Id, *> | vis.DataSet

CollectionTypeString

Most of tiddlymap's select statements allow the user to choose one of the specified output types, depending on what is most suitable.

The output types relate to the different types of EdgeCollection and NodeCollection.
