<static> hasOwnProp(obj, key)

I use this method on all objects that I didn't create myself.

Why this? Well,

  1. How do I know if the object was created via {} or utils.getEmptyMap()? If the latter is the case, hasOwnProperty() doesn't exist.
  2. When the object is used as hashtable, hasOwnProperty could be overridden.
