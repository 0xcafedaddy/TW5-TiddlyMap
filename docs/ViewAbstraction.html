Class: ViewAbstraction

ViewAbstraction

new ViewAbstraction(view, isCreate)

This class abstracts the various pieces that together make up the view such as map, edge filter, node filter, config etc. If {@code isCreate} is not specified, the viewAbstraction will only represent the view and not create it or any missing part of it.

True if the view should be created and override any existing view, false otherwise.

getEdgeStoreLocation() → {string}

This method tells the caller where to find the edges of this view.

If this view exists and is currently run in private mode, all edges have to be read from and stored under the view's root. If not run in private mode, the global stores have to be used. The decision which prefix to use is determined by the state of the views private_edge_mode configuration.
