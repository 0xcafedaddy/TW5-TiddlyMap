Class: EdgeType

EdgeType

new EdgeType(type)

This class is used to abstract edge types. It facilitates inter alia the parsing of style information, the translation of type names into actual type data or the persistance of edge type data.

Parameters:
Name Type Description
type string | EdgeType

Either the edge type id (name) or a tiddler reference denoting the type or an EdgeType object (that is directly bounced back). If the id can be translated into a tiddler object that resides in the edge type path, then its data is retrieved automatically.

loadDataFromTiddler(edgeType, isAllowOverideId)

Retrieve all data from the tiddler provided. If a shadow tiddler with the same id exists, its data is merged during the load process.

Parameters:
Name Type Argument Default Description
edgeType *

The edge-type containing the data. The edgeType parameter may be a string denoting the type's id, a path (tiddler title), a $tw.Tiddler object or an EdgeType object.
isAllowOverideId boolean <optional>
 false

If the provided tiddler has an id field set, setting isAllowOverideId to true will cause the currently held id to be overridden.

persist(tRef, isPrettifyJSON)

Store the edge-type object as tiddler in the wiki. If the title property is not provided, the default edge-type path prefix will be used with the edge-type id appended. The style data is written as JSON into the text field.

Parameters:
Name Type Argument Description
tRef string <optional>

If tRef is provided, the edge-type data will be written into this tiddler and the id property is added as extra field value. Do not use this option if you want the system to recognize the edge type. This is only for dumping purposes.
isPrettifyJSON boolean <optional>

True, if any json data should be stored in a prettified way, false otherwise.
