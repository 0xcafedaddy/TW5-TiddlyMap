add(TiddlerReference, callback, deleteOnCall)

The callback mechanism allows to dynamically listen to tiddler changes without hardcoding a change-check for a tiddler name in the refresh function.

Parameters:
Name Type Argument Default Description
TiddlerReference <optional>

tRef - A tiddler whose change triggers the callback.
callback function

A function that is called when the tiddler has changed.
deleteOnCall boolean <optional>
 true

True if to delete the callback once it has been called, false otherwise.

handleChanges()

this method has to be implemented at the top of the refresh method. It checks for changed tiddlers that have registered callbacks. If deleteOnCall was specified during registration of the callback, the callback will be deleted automatically.
